# ОГЭ • Штаб подготовки (React + Vite)

Это **готовый React-проект**, собранный из вашей страницы и исправленный так, чтобы:
- всё работало как настоящее React-приложение (без Babel в браузере и без `import` внутри обычного `<script>`);
- был **режим DEV** на localStorage (можно тестировать без Firebase);
- был **админ‑режим** (код задаётся в `.env`);
- у каждого ученика свой прогресс (по **UID**, который сохраняется в браузере).

## Быстрый старт
1) Установите **Node.js LTS**  
2) В папке проекта:
   ```bash
   npm install
   npm run dev
   ```
3) Откройте адрес из консоли (обычно `http://localhost:5173`).

## Сборка для GitHub Pages
```bash
npm run build
```
Готовая папка будет `dist/`. Её содержимое можно заливать на GitHub Pages.

## Настройки (необязательно)
Скопируйте файл `.env.example` → `.env` и отредактируйте:
- `VITE_DEV_MODE=true|false`
- `VITE_ADMIN_CODE=...`
- Firebase‑поля `VITE_FIREBASE_...` (если включаете прод)

Если **Firebase не настроен**, проект автоматически работает в DEV‑режиме (localStorage).

## Где менять контент
`src/data.js` — уроки, тесты, материалы и видео.

## Важно про "личный кабинет"
Сейчас прогресс хранится в браузере ученика:
- если ученик заходит с другого устройства/браузера — это будет **другой прогресс** (это нормально для DEV).
Чтобы прогресс был единый на всех устройствах, включите Firebase (см. `.env.example`).

Удачи! Если захотите — могу подстроить структуру под ваш реальный список уроков/игр/заданий.
